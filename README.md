# JWT-FASTAPI

A production-ready FastAPI authentication system with JWT tokens, HttpOnly cookies, and PostgreSQL database. Built with uv for lightning-fast package management.

## Features
✅ Secure Authentication - JWT tokens with refresh token rotation

🔒 HttpOnly Cookies - Tokens stored securely, inaccessible to JavaScript

🔄 Token Refresh - Automatic token renewal without user interaction

🗄️ PostgreSQL Database - Robust, production-ready database

🔐 Password Hashing - bcrypt for secure password storage

⚡ UV Package Management - Blazing fast dependency resolution

📱 Responsive Design - Clean HTML templates with CSS

🚦 Rate Limiting - Protect against brute force attacks

📝 SQLAlchemy ORM - Type-safe database operations

🧪 Alembic Migrations - Easy database schema versioning

🔧 Environment Config - 12-factor app design

## Tech Stack
```
Technology	Purpose
FastAPI	Web framework
PostgreSQL	Database
SQLAlchemy	ORM
Alembic	Database migrations
python-jose	JWT token handling
passlib	Password hashing (bcrypt)
jinja2	HTML templating
python-dotenv	Environment variables
uvicorn	ASGI server
uv	Package management

```
